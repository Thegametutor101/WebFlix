Exigence minimal pour Webflix


Condition minimal

Résolution supporté

Matériel minimal

Vous pouvez utiliser le lecteur HTML5 de Webflix ou le plug-in Silverlight pour visionner le

contenu du site sur les appareils suivants :
Ordinateurs PC exécutant Windows XP ou une version ultérieure
Ordinateurs Mac dotés d'un processeur Intel exécutant OS X Tiger (v10.4.11) ou une version ultérieure
iPads exécutant iPadOS 13.0 ou une version ultérieure
Consultez les exigences complètes ci-dessous pour HTML5, Silverlight 4 et Silverlight 5.

Résolution :

Google Chrome, Windows, Mac et Linux : jusqu'à 720p
Chrome OS : jusqu'à 1080p
Microsoft Edge : jusqu'à 4K*
Internet Explorer : jusqu'à 1080p
Mozilla Firefox : jusqu'à 720p
Opera : jusqu'à 720p
Safari : jusqu'à 1080p sur macOS versions 10.10 à 10.15
Safari : jusqu'à 4K sur macOS version 11.0 ou ultérieure
*La diffusion en 4K requiert une connexion conforme à la norme HDCP 2.2 vers un écran capable d'afficher une image 4K, un processeur Intel Core de 7e génération, et les dernières mises à jour de Windows. Consultez le fabricant de votre système pour en vérifier les spécifications.
